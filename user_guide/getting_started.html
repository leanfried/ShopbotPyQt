Software installation

Software notes

  • This software was written for Windows. Certain elements, like folder opening buttons and the system that tracks whether the pressure should be on or off during prints, will break outside of Windows.
  • This software is currently in development. Please report issues through Github or by emailing the maintainer.

Getting started

  1. Install Python
    • For example, install Miniconda
  2. Install the Shopbot control software
    • Open the Shopbot control software SB3.exe and load the settings for your Shopbot.
    • By default, the Shopbot control software shows 4 output flags, but it allows 12. To increase the number of visible output flags for signaling additional devices (e.g. pressure channels, lights, etc.), go into Move/Cut Mode. Once in Move/Cut - right click in the input/output section of the red position window. Here you can set the number of inputs/outputs displayed up to 12 in the red window
  3. Download the software from Github. Because this software is continually updated, you can keep the most updated version of the code by cloning the repository through Github. For beginners:
    • Download Github desktop
    • Create a Github account
    • In Github desktop, clone the ShopbotPyQt repository
    • To update your software to the latest version, click "Fetch Origin" in Github Desktop at any time.
  4. Install Python packages listed in requirements.txt.
  5. Launch the GUI.
    • In Miniconda: python "MyGithubPath\ShopbotPyQt\pythonGUI\sbgui.py", replacing "MyGitHubPath" with your full github path.
    • In a Jupyter console from the pythonGUI folder: run sbgui.py
  6. OPTIONAL: If you are working with .stl files or other 3D model files, install a slicer.
    • For example, install Ultimaker Cura. Save it to the system's 'program files' folder
    • Create a free ultimaker account
    • Once downloaded and opened, go to the setting tab and add printer. Under the 'add a non-networked printer', there should be a 'custom' option with a drop down menu. Click on the 'Custom FFF printer' and rename it to your preference.
    • A 'machine setting' window should pop up, where you will place corresponding variables to the printer. For the embedded 3D printing configuration using a 25 mm x 25 mm x 75 mm bath and a 20 gauge needle:

    • Set the print resolution to extra coarse - 0.6 mm and unclick the Adhesion box and Support box in the same window
    • Support and skirts are not necessary in embedded 3D printing. If printing into a bath, within the print settings 'Custom', make certain the 'Infill' section's density is set to 0.0, the 'Support' section's Generate Support box is unchecked, and the 'Build Plate Adhesion' type is set to None.
    • Note that the Cura slicer creates a 1/3 inch (8.47 mm) margin at the edges of the print bed where it will not allow you to place material. If you wish to print closer to the walls, you can increase the X/Y/Z measurements.
    • When creating an STL file through Onshape, create measurements in millimeters, even if it states in inches. Cura assumes all measurements are in millimeters.
